# Assignment-5-Fun-with-JavaScript
Users can type text into the text field and use the controls to change it's styling. The “Bigger!” button increases the text size, “FancyShmancy” makes the text bold, blue, and underlined, and “BoringBetty” returns the text to it's orginal styling (excluding size). The “Moo” button uppercases the text and adds “-Moo” to the end of each sentence!
